Declaring variables

To declare a number type

uimhir variableName = 5

To declare a string type

focal variableName = DiaDuit

To declare a boolean type as true

realtacht variableName = fior

To declare a boolean type as false

realtacht variableName = breagach

  • Numbers in Draiocht are semi-dynamically typed
  • If you type Uimhir num =5.5, Draiocht will make this a float
  • Similarly if you type a very large number Uimhir num =2147483648, Draiocht will make this an UimhirMor object
    • If else statements

    To write an if

    ma conidtion codeblock

    To write an if else

    ma conidtion codeblock eile codeblock

    To write an else if

    má eile conidtion codeblock

  • "má"" is the irish for if
  • "eile" is the irish for else
  • "má eile" is the irish for if else
    • While loops

    To write a while loop

    fad conidtion codeblock

  • fad is the irish for while
    • For loops

    To write a for loop

    do conidtion codeblock

  • "do" is the irish for "for"
    • Functions

    feidhm functionName(input value if any)
